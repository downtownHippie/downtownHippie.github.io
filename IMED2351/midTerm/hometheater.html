Using the PlayStation to play DVDs and stream NetFlix

  • Turn everything on
    1. the Playstation
    2. the Integra Receiver
    3. the TV
  • Select GAME for input on the receiver
  • To watch a DVD just insert it
  • To stream Netflix or HuluPlus
    1. Using the Playstation remote
    2. Navigate across to TV/Video Services
    3. Navigate down to the desired service
    4. Select Netflix or HuluPlus to stream that service

the Playstation's fan gets loud, turn if off when you're done

  • touch and hold the power icon on the Playstation to power it off

Using the Roku

  • Turn every thing on
    1. the Integra Receiver
    2. the TV
  • Select AUX for input on the receiver
  • Sometimes the Roku is finiky about being in sleep mode for long periods of time
    1. Remove the skinny power cord from the back of the Roku
    2. Slowly count to 5
    3. Plug the skinny power cord back into the Roku

If there is no video but there is audio

  • The TV source has been switched from HDMI1 to TV
  • Using the TV remote
    1. Press the source button
    2. Navigate to HDMI1
    3. If the receiver is not powered on HDMI1 will not be selectable

To view photos from a USB hard drive or thumb drive

  • Insert drive or cable into USB port on left side of TV
  • Using the TV Remote
    1. Press the source button
    2. Navigate down to USB
